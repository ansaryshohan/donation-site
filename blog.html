What is the Document Object Model (DOM)?

Document Object Model is a process to access HTML pages with Javascript. DOM makes a html page functional and interactive. DOM is a tree like structure for accessing html contents.

How do you select an element from the DOM?

We can select an element in 5 ways by using

  1. 1. getElementsByClassName
  2. 2. getElementsByTagName
  3. 3. getElementById
  4. 4. querySelectorAll
  5. 5. querySelector
What is event delegation in the context of the DOM, and why is it useful?

Event Delegation means is to handle the event from the parent. it means giving a task to other element. Here we will declare the event on the parent but it will synchronously if we interact with the children of the parent.

How do you manipulate an element's attributes and styles using the DOM?

1. We can manipulate element's attributes with setAttribute() and removeAttribute()

2. We can manipulate element's styles with document.getElementById("").style.property="style-value"
